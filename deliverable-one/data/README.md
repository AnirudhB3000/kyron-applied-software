# Data Sources

This directory stores the large NPPES source files and reference documents used by the deliverable pipeline.

## Notes

- The bulk data files in this directory are ignored from version control because of their size.
- The source for the NPPES bulk files is the CMS download page:
  - https://download.cms.gov/nppes/NPI_Files.html
